# Admin task content generation

Admin AI tasks generate approval-gated drafts for reply suggestions, conversation summaries, FAQ mining, post captions, story text, campaign messages, product announcements, recovery messages, and product comparison responses.

Rules:

* Drafts never auto-publish in v1.
* Output is grounded in supplied catalog, policy, and conversation context.
* Prompt/model/schema metadata is stored in output metadata.
* Admins can regenerate, edit, approve, reject, and copy drafts.
